uint8_t * Comm::read_data ( )

This function is used to read the serial interface received data.

Returns
a pointer to the serial interface received data This function return value may be either a pointer to the actual data or, to avoid making this function blocking, if any data has been received, a pointer to an array filled with 0x00. The returned data length will be exactly the same as @refe Comm packet size.
